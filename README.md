## Setup
Ensure ArangoDB & Redis are running
Modfy ```congif.example.js``` to be ```config.js``` and replace the ```"FILL_ME_IN"``` with your ArangoDB's proper password

For local testing
#### Inital setup 
```
$ cd ../photoGallery-service
$ npm install
$ npm start
```
